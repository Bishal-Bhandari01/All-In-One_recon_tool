# Requirements:
  <h2>You need before installing using this python script.</h2>
<ol>
  <li>Install python3</li>
  <li>Install sockets</li>
  <li>Install requests</li>
</ol>

# Usage:
```
for Port scanning:
    ```python3
    python3 allinone.py -net <ip/domain name>
    ```
for sub-domain:
    ```python3
    python3 allinone.py -subs <domain name>
    ```
```
